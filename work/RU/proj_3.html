Это мой личный таск менеджер для тасков с короткой продолжительностью.

Используемые технологии : ReactJS, ECMAScript6, CSS3, HTML5, SASS, Balsamiq Mockups.

Также я использовал Redux для хранения состояний, React-router для роутинга и Babel для транспиляции.
