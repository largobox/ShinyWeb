О проекте:

Это мой личный таск менеджер для оптимизации моей деятельности.

Описание работ:

Всё реализовано только на ReactJS. Также я использовал Redux для хранения состояний, React-router для роутинга и Babel для транспиляции.

Стэк технологий:

ReactJS, ECMAScript6, Babel, Webpack, HTML5, SASS.

Статус:

С удовольствием пользуюсь своим таск менеджером.
