О проекте:

На первый взгляд, обычный блог. За исключением одной особенности: владелец такого блога не платит абсолютно ничего за его содержание. При этом отсутствует какая-либо сторонняя реклама. Такой результат достигается благодаря сервисам Github, также блог оснащён функциями:

  • Поиска по статьям
  • Поиска по тэгам
  • Авторизации через соц. сети. При желании набор текущих соц. сетей можно расширить
  • Комментирования статей

Описание работ:

Back-end написан на Ruby on Rails. Front-end реализован на Jekyll. Взаимодействие с сервером происходит посредством Ajax.

Стэк технологий:

PostgreSQL, Ruby on Rails, Jekyll, SASS, HTML5, AJAX, JavaScript, GithubPages, Heroku
