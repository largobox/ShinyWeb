About project:

At first glance, the usual blog. Except for one feature: the owner of this blog doesn`t pay absolutely nothing. At the same time, there is no third-party advertising. It`s worth noting the following features:

  • Search by posts
  • Search by tags
  • Authorization through social networks. If desired, the set of current social networks can be expanded
  • Commenting on articles

Description of development:

Back-end was written in Ruby on Rails. Front-end is implemented on Jekyll. Interaction with the server occurs through Ajax.

Stack of technologies:

PostgreSQL, Ruby on Rails, Jekyll, SASS, HTML5, AJAX, JavaScript, GithubPages, Heroku
